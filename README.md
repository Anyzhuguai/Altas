## 🚀 核心功能实现 (Key Features)

### 1. 复杂路径识别算法 (Path Recognition)
基于 8 路 I2C 巡线传感器的原始数据，通过位运算进行模式匹配，实现了以下逻辑：
*   **〰️ 弧线/S弯巡线**: 
    *   不再是简单的“左转/右转”，而是计算传感器偏离中心点的**误差值 (Error)**。
    *   使用 PD 算法（比例+微分）计算转向差速，使车辆在弧线过弯时更加丝滑，减少震荡。
*   **➕ 十字路口 (Crossroad)**:
    *   **判定逻辑**: 检测到传感器全黑 (`11111111`) 或中间宽范围黑线。
    *   **处理策略**: 状态机记录路口特征，根据预设导航序列（如：第1个路口左转，第2个直行）执行动作。
*   **T 丁字路口 (T-Junction)**:
    *   **判定逻辑**: 检测到全黑线，且前瞻逻辑判断前方无路。
    *   **处理策略**: 强制减速并进入直角转弯模式。

### 2. 直角转弯逻辑 (90° Turn)
*   针对 90 度急弯，仅靠差速调节容易冲出赛道。
*   **实现方案**: 当识别到直角特征时，暂停巡线 PID，进入**开环/半闭环转弯模式**（锁定一侧电机，另一侧旋转），直到传感器重新捕捉到黑线后再切回巡线模式。

### 3. [WIP] 速度环 PID 控制 (Current Work)
*   **背景**: 之前仅有方向环 PID（控制转向），电量下降或负载变化时速度不稳定，过弯容易因速度过快冲出。
*   **解决方案**: 引入电机自带的**编码器 (Encoder)** 反馈。
    *   **串级 PID 架构**: 
        *   **内环**: 速度环 (控制电机转速恒定)。
        *   **外环**: 转向环 (计算差速)。
    *   **目标**: 实现“直道加速，弯道自动减速”的智能控制。
